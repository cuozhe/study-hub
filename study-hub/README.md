# StudyHub - 智能学习助手

一个整合费曼技巧、艾宾浩斯复习法、番茄工作法、康奈尔笔记法、西蒙学习法的综合性学习辅助工具。

## ✨ 功能特点

### 五大学习法有机整合
1. **番茄工作法** - 25分钟专注计时，帮助你保持高效学习节奏
2. **西蒙学习法** - 追踪连续学习天数和总学习时长，设定每日目标
3. **康奈尔笔记法** - 三栏式笔记模板（线索栏/笔记栏/总结栏）
4. **费曼技巧** - 学完必须用自己的话输出，强制检验理解程度
5. **艾宾浩斯复习法** - 自动生成1/2/4/7/15/30天复习计划

### 六大数据视图
- **📅 日历视图** - 一眼看尽学习和复习安排
- **📊 统计视图** - 学习时长、番茄钟数、连胜天数等数据可视化
- **⏰ 时间线视图** - 按时间顺序回顾学习历程
- **🗺️ 知识地图** - 可导入的学习路径，监控学习进度
- **📝 笔记墙** - 所有康奈尔笔记的卡片流展示
- **🔄 复习队列** - 根据艾宾浩斯曲线管理复习任务

### 博客导出
一键复制 Markdown 格式的学习记录，包含：
- 今日学习统计
- 康奈尔笔记
- 费曼输出
- 艾宾浩斯复习计划

## 🚀 快速开始

### 1. 部署到 Vercel

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建
npm run build
```

### 2. 使用流程

1. 进入"地图"视图，选择或导入学习路径
2. 选择今日学习的主题，点击"开始学习"
3. 开启番茄钟，完成25分钟专注学习
4. 填写康奈尔笔记（线索栏/笔记栏/总结栏）
5. 完成费曼输出（至少30字）
6. 系统自动生成艾宾浩斯复习计划
7. 一键复制 Markdown 到博客

## 📝 学习路径导入

你可以导入自定义的学习路径（JSON 格式）：

```json
{
  "name": "网络工程师（中级）",
  "description": "软考学习计划",
  "subjects": [
    {
      "name": "计算机基础",
      "topics": [
        { "name": "硬件基础", "status": "available" },
        { "name": "操作系统", "status": "locked" }
      ]
    }
  ]
}
```
